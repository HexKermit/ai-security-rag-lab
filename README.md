# AI Security RAG Lab

A beginner-to-intermediate AI security project that explores vulnerability retrieval using structured data, hybrid matching, and embedding-based semantic search.

## Features

- Structured vulnerability dataset in JSON
- Modular Python project layout
- Semantic search with sentence-transformers
- Input normalization for cleaner queries
